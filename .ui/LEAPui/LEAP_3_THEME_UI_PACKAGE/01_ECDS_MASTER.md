# LEAP Enterprise Cognitive Design System (ECDS)

## Objective
Create an enterprise cognitive workspace that feels:
- Trustworthy
- Authoritative
- Calm
- Premium
- Safe for decision making

## Design inspirations
- OpenAI
- Apple HIG
- Linear
- Notion
- Stripe
- Palantir Foundry

## Universal rules
- Rounded corners: 12–20px
- Matte surfaces
- Soft shadows
- Inter/SF Pro/Geist typography
- Mathematical formulas rendered as readable math, never code
- Governance alerts only in right sidebar
- Preserve traceability and auditability
