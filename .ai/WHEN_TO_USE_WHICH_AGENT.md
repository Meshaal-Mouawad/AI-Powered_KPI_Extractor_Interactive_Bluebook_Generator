# WHEN_TO_USE_WHICH_AGENT.md

## Gemini
Use for:
- repository analysis
- architecture understanding
- documentation
- large context review

Avoid:
- expensive repeated debugging loops

---

## Codex
Use for:
- hard bugs
- deep root-cause analysis
- rendering problems
- architecture validation

Avoid:
- routine implementation tasks

---

## Copilot
Use for:
- implementation
- boilerplate
- refactoring
- IDE assistance

---

## Human
Required for:
- business truth
- visual verification
- governance approval
- final acceptance
